# Bilibili-barrage-machine
一个把bilibili弹幕通过rcon转发到我的世界聊天框

将B站直播间弹幕和礼物实时转发到《我的世界》服务器的工具，通过RCON协议实现跨平台互动。

## 功能特性

- 🎯 **双向互动**：实时同步B站弹幕到游戏内聊天栏
- 🎁 **礼物播报**：自动感谢观众赠送的礼物
- 🛡️ **安全连接**：使用官方API和RCON协议
- 🖥️ **图形界面**：简单易用的配置窗口
- 🌈 **彩色消息**：支持Minecraft颜色代码格式化
